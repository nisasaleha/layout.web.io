Pengenalan Layout Responsif untuk halaman web - Layout adalah penataan elemen halaman yang umumnya ada pada perancangan suatu suatu sistem informasi maupun web. Layout Responsif adalah salah satu layout yangs sering diterapkan pada halaman website. Layout responsif memungkinkan halaman web menyesuaikan diri dengan ukuran perangkat yang mengaksesnya, berikut ini adalah tampilan web yang sama ketika dibuka atau diakses menggunakan perangkat yang berbeda:.

Pentingkah layout responsif? Semakin hari semakin banyak orang menggunakan perangkat mobile atau browsing, seperti menggunakan smartphone dan tablet untuk setiap tugas atau pekerjaan sehari-hari. Pekerjaan yang dulunya hanya dilakukan melalui personal komputer(dekstop&laptop) sekarang hampir semua sudah bisa digunakan juga melalui perangkat mobile.
